KEY POINT

  • link&import

1. 구글 폰트에 들어간다
2. 원하는 폰트를 고른다
3. 네모난 박스를 누른다
4. 원하는 사이즈를 고른다
5. import를 누른 후 복사해서 가져온다
6. css파일에 폰트패밀리를 복붙해서 넣어준다

  • selector

    id 사용배경: css를 주고 싶은 특정한 태그 1개가 있을때
    css 적용방법:<스타일> #id이름{적용하고 싶은 css}
    class 사용배경:비슷한 특성을 주고 싶은 태그가 여러개일때
    css 적용방법: <스타일> .class이름{적용하고 싶은 css}

    가상클래스

    class 나 id 태그 이름으로 태그를 지정했다면 행동의 변화로도 css 효과를 줄 수 있다
    종류: active, hover,visited,link,focus
    active:누르고 있는 순간
    hover:누르지 않고 마우스를 올리고 있을 때 클릭
    visited:마우스로 누르고 나서
    link: 마우스 클릭 전
    focus: 작성을 위해 클릭한 상태 ex: 커서가 깜빡할 때
